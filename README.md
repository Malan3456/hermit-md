#### RUN ON VPS/UBUNTU/WINDOWS

1. Install NodeJs,ffmpeg
2. Installation
   ```
   $ npm i -global pm2
   $ git clone https://github.com/A-d-i-t-h-y-a-n/hermit-md.git
   $ cd hermit-md
   $ npm install
   ```
3. Configuration
   ```
   $ echo "VPS = true
     SESSION_ID = null
     AUTH_FILE = session
     SUDO = null
     PREFIX = .
     MODE = private
     LOG_MSG = true" > config.env
   ```
- Start
  ```
  $ npm start
  ```
- Stop
  ```
  $ pm2 delete hermit-md
  ```
